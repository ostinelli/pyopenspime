__init__(self, osid_or_osid_path, osid_pass='', server='', port=None, try_reconnect=60, rsa_pub_key_path='', rsa_priv_key_path='', rsa_priv_key_pass='', rsa_key_cache_path='cache', cert_authority='', accepted_cert_authorities_filepath='certification-authorities.conf', log_callback_function=None)
(Constructor)

source code 

Initializes a Client.

Parameters:
  • osid_or_osid_path (str) - The full OSID of the client. If an OpenSpime configuration package is found, this is the only parameter that is needed to initialize the Client.
  • osid_pass (str) - The full OSID password. Taken from the OpenSpime configuration package if found.
  • server (str) - The server address. Defaults to the OSID domain.
  • port (int) - The server port. Defaults to 5222.
  • try_reconnect (int) - Reconnects if connection drops. Set to 0 if no reconnect, otherwise integer expresses interval to reconnection trials in seconds. Defaults to 60.
  • rsa_pub_key_path (unicode) - The path to the RSA public key .pem file. Taken from the OpenSpime configuration package if found.
  • rsa_priv_key_path (unicode) - The path to the RSA private key .pem file. Taken from the OpenSpime configuration package if found.
  • rsa_priv_key_pass (unicode) - The RSA private key .pem file password. Taken from the OpenSpime configuration package if found.
  • rsa_key_cache_path (unicode) - The path to cached public RSA keys of entities. Defaults to /cache.
  • cert_authority (unicode) - The client's certification authority. Taken from the OpenSpime configuration package if found.
  • accepted_cert_authorities_filepath (unicode) - The path to the file of accepted certification authorities. Defaults to file certification-authorities.conf in root folder.
  • log_callback_function (function) - Callback function for logger. Function should accept two parameters: unicode (the log description) and integer (the verbosity level - 0 for error, 1 for warning, 2 for info, 3 for debug).

send_stanza(self, stanza, to_osid='', encrypt=None, sign=None, extname='', callback_success=None, callback_failure=None, callback_timeout=None, timeout=60)

source code 

Sends out a stanza.

Parameters:
  • stanza (pyopenspime.xmpp.protocol.Protocol) - The stanza to be sent.
  • to_osid (unicode) - The recipient of the message.
  • encrypt (boolean) - If encryption is requested, set to True. Defaults to False.
  • sign (boolean) - If signature is requested, set to True. Defaults to False.
  • extname (str) - If stanza is representative of a request, the extension name of the request send out.
  • callback_success (function) - Callback function called when a 'result' response is received. This parameter is ignored if the stanza being sent is not an <iq/> stanza.
  • callback_failure (function) - Callback function called when a 'error' response is received. This parameter is ignored if the stanza being sent is not an <iq/> stanza.
  • callback_timeout (function) - Callback function called when no response is received after the timeout period. This parameter is ignored if the stanza being sent is not an <iq/> stanza.
  • timeout (int) - If a callback_timeout function has been specified, this parameter specifies the timeout in seconds after which the callback_timeout function is called if no response is received. This parameter also specifies the time life of the callback_success and callback_failure functions, after which their handler will be removed. This parameter is ignored if the stanza being sent is not an <iq/> stanza.
Returns: unicode
ID of sent stanza.

on_disconnect(self)

source code 

Event raised on a disconnection to the XMPP server. This one does nothing, should be overriden in derived classes.

Note that reconnection attempts are handled automatically, therefore any blocking on_disconnect() derived function will therefore compromise such attempts.
