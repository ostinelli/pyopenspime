Initializes a Client.

Parameters:
  • osid_or_osid_path - The full OSID of the client. If an OpenSpime configuration package is found, this is the only parameter that is needed to initialize the Client.
  • osid_pass - The full OSID password. Taken from the OpenSpime configuration package if found.
  • server - The server address. Defaults to the OSID domain.
  • port - The server port. Defaults to 5222.
  • try_reconnect - Reconnects if connection drops. Set to 0 if no reconnect, otherwise integer expresses interval to reconnection trials in seconds. Defaults to 60.
  • rsa_pub_key_path - The path to the RSA public key .pem file. Taken from the OpenSpime configuration package if found.
  • rsa_priv_key_path - The path to the RSA private key .pem file. Taken from the OpenSpime configuration package if found.
  • rsa_priv_key_pass - The RSA private key .pem file password. Taken from the OpenSpime configuration package if found.
  • rsa_key_cache_path - The path to cached public RSA keys of entities. Defaults to /cache.
  • cert_authority - The client's certification authority. Taken from the OpenSpime configuration package if found.
  • accepted_cert_authorities_filepath - The path to the file of accepted certification authorities. Defaults to file certification-authorities.conf in root folder.
  • log_callback_function - Callback function for logger. Function should accept two parameters: unicode (the log description) and integer (the verbosity level - 0 for error, 1 for warning, 2 for info, 3 for debug).
